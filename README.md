# Techinika E-Learning Platform

Welcome to the Techinika E-Learning Platform repository! This project is built with Next.js and Tailwind CSS.

## Overview

Techinika E-Learning Platform is a dynamic and interactive online learning platform that aims to provide a seamless experience for students and instructors. Leveraging Next.js for efficient server-side rendering and Tailwind CSS for responsive styling, this platform is designed to make online education engaging and effective.

## Features

- User-friendly course listings and details.
- Secure user authentication and account management.
- Dynamic routing for enhanced navigation.
- Responsive design optimized for various devices.
- ...

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/techinika/school-fe.git
   ```

2. Navigate to the project directory:

   ```bash
   cd school-fe
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000` to see the project.
